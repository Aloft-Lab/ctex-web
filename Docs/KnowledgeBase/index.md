## TeX 及其扩展

TeX 的扩展，又称 TeX 格式。最著名也是最常用的 TeX 格式就是 LaTeX 。最基本的 TeX 程序只是由一些很原始的命令组成，对用户不够友好。TeX 也允许用这些原始命令定义一些更复杂的高级命令，也就是 TeX 格式，对用户更加友好。
